# Python UNO Game

A simple **Python implementation of UNO**, supporting both human and AI players. The game includes special cards like Skip, Reverse, Draw Two, Wild, and Wild Draw Four, and handles UNO declarations and penalties.

---

## Features

- Play with **2â€“4 players**, including **AI opponents**.
- Standard **UNO deck** with colors, numbers, and action cards.
- Handles **special cards effects**:  
  - Skip  
  - Reverse  
  - Draw Two  
  - Wild  
  - Wild Draw Four
- **UNO declaration** and penalties for forgetting to declare.
- AI players with basic decision-making (80% chance to remember UNO).
- Win condition prevents finishing on a special card.
- Dynamic turn order with **reversals**.
